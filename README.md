# Workday Autofill
A chrome add-on to make filling in Workday applications less arduous

What's Included:
Core Files:

manifest.json - Extension configuration
popup.html/css/js - Beautiful UI for uploading and managing your resume
content.js - Auto-fill logic that runs on Workday pages
background.js - Service worker for background processing
Extension icons (16px, 48px, 128px)

Bonus Files:

README.md - Comprehensive documentation
QUICKSTART.md - 5-minute setup guide
sample-resume.txt - Example resume for testing

✨ Key Features:

Resume Upload - Drag & drop or click to upload PDF, DOC, DOCX, or TXT
Smart Parsing - Automatically extracts name, email, phone, experience, education, skills
Editable Data - Review and edit parsed information
One-Click Auto-Fill - Fills Workday forms instantly
Privacy First - All data stays in your browser
Visual Feedback - Success notifications and status updates

📦 Download Your Extension:
View extension folder | Download ZIP
🚀 Installation (2 minutes):

Extract the ZIP file
Open Chrome/Brave and go to chrome://extensions/
Enable "Developer mode" (top-right toggle)
Click "Load unpacked" and select the workday-autofill folder
Click the extension icon and upload your resume!

The extension will automatically detect Workday pages and show a green checkmark badge when active. Check out the QUICKSTART.md for a detailed walkthrough!
